# PROJECT: MAIL MONSTER PRO (Baseline v2.5 -> v2.5.1 Update)

## 1. 현재 구현 상태 (v2.5 완료 내역)
- **UI**: 다중 프로필 사이드바 기반 계정 관리 시스템.
- **Editor**: pywebview 기반 WYSIWYG 에디터 및 테스트 발송 기능.
- **Security**: 구글 시트 기반 중앙 통제 블랙리스트 및 자동 업데이트 시스템.
- **Statistics**: 글로벌 발송 카운터 및 엑셀 리포트 추출.

## 2. v2.5.1에서 해결해야 할 기술적 과제
1. **Gmail 세션 타임아웃 오류**: 
   - 현재 로직이 '서버 연결 -> 메일 조립' 순서로 진행되어 구글 서버에서 `Server not connected` 오류가 빈번히 발생함.
2. **비효율적 중복 발송 방지**:
   - 동일 수신자에게 '다른 내용(템플릿)'을 보낼 때도 차단되는 이슈 존재.
   - `email` + `template_name`을 결합한 유연한 중복 검사 로직 확립 필요.
3. **로그 가독성 이슈**:
   - 발송 성공 로그에 수신처(업체명)가 가끔 누락되어 확인이 어려운 현상 수정.
